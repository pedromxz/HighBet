Criar uma pasta para o projeto e rodar no cmd:

# Inicia o composer;
    composer init

# Preencher os dados solicitados

# Alterar o arquivo composer para:
     "psr-4": {
            "App\\": "app/"
        }

# Criar uma pasta public com um index.php dentro

# Iniciar o servidor do PHP
    php -S localhost:8000 -t public 


## QUANDO ABRE O PROJETO EM UM NOVO PC
    composer install

## SE DER ERRO DE ROTA (404)
    composer dump-autoload
